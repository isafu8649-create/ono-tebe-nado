Проектная работа: "Оно тебе надо"
Аукцион вещей, в которые никто не верил

Описание
Верстка лендинга для аукциона необычных вещей по макету из Figma. Проект выполнен в рамках обучения на Яндекс Практикуме.

Документация проекта
Особенности реализации
Семантическая верстка с использованием HTML5 тегов

Переиспользование CSS-классов 

Адаптивная сетка на CSS Grid и Flexbox

Соответствие макету Pixel Perfect

Метод прогрессивного джипега в разработке

Валидная разметка

Оптимизированные изображения

Секции страницы
Шапка (Header) - меню, логотип, контакты

Главная секция (Cover) - заголовок, описание, кнопка

Лоты (Lots) - три карточки с лотами аукциона

Об аукционе (About) - информация о проекте

Подвал (Footer) - контакты, меню, социальные сети

Как открыть проект
Склонируйте репозиторий:

bash
git clone git@github.com:isafu8649-create/ono-tebe-nado.git
Откройте файл index.html в браузере

Требования
Браузер с поддержкой HTML5 и CSS3

Минимальная ширина экрана: 1100px
